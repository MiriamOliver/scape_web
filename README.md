# scape_web
## Para Back (Posicionarte en la carpeta back)

Actualizar dependencias:

    npm install
    
Lanzar el servidor:

    nodemon app
    
## Para Front (Posicionarse en la carpeta front)

Actualizar dependencias:

    npm install

Iniciar el cliente:

     ng serve -o

## Base de Datos
Crear una base de datos llamada scape_web <br>
Importar el archivo scape_web.sql <br>
